# Anime Score Analysis using Pandas

This project analyzes anime ratings by grouping anime types and calculating their average scores using Pandas.

The goal is to understand which anime categories tend to receive higher ratings.

---

# 📊 Project Overview

The project:
- Groups anime by type
- Calculates average scores
- Sorts results from highest to lowest
- Visualizes results using charts

---

# 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
